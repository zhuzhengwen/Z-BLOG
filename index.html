⚙️ 设置 GitHub Token

Token 仅保存在你的浏览器中，不会上传到任何地方。
设置后 API 限额从 60次/小时 提升至 5000次/小时。

⚠️ 当前 IP 已触发频率限制，设置 Token 后即可继续使用。

前往 GitHub → Settings → Developer settings → Personal access tokens → 勾选 public_repo 生成
